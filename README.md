# ABR Website Monorepo

This project is now organized as a monorepo with separate frontend and backend apps.

## Structure

- `frontend` - React + Vite storefront UI
- `backend` - Node.js + Express API (`/api/products`, `/api/health`)

## Run

```bash
npm install
npm run dev
```

Frontend runs on `http://localhost:5173` and backend runs on `http://localhost:4000`.

## Build

```bash
npm run build
```
